# 📊 Visualização de Notas com Matplotlib

Este é um projeto simples em Python que utiliza a biblioteca **Matplotlib** para gerar um gráfico de barras. O objetivo é demonstrar como criar uma visualização de dados clara, estilizada e de fácil leitura.

## 🚀 Tecnologias
* **Python**
* **Matplotlib**

## 📝 Descrição do Código
O script gera um gráfico de barras comparando a nota média de 5 alunos. Foram aplicadas personalizações como:
* Cores customizadas (`skyblue` com bordas `navy`).
* Grade de fundo (`grid`) para facilitar a leitura dos valores.
* Ajuste de limites do eixo Y ($0$ a $100$).
* Títulos e rótulos de eixos formatados.
